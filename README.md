# ¡Bienvenido a mi baúl de código reutilizable! 🎉

Este repositorio es una colección personal de plantillas, fragmentos y configuraciones que uso para acelerar mi flujo de trabajo. Aquí guardo desde estructuras base para proyectos hasta pequeños snippets que utilizo con frecuencia, para no tener que reinventar la rueda cada vez.

## 📋 Estructura

El repositorio está organizado por tecnología y propósito para facilitar la búsqueda.

## ✨ Lo que encontrarás aquí

- ✅ Boilerplates limpios y listos para usar.
- ✅ Fragmentos de código (snippets) para tareas repetitivas.
- ✅ Configuraciones de herramientas (ESLint, Prettier, etc.).
- ✅ Componentes de UI sencillos y desacoplados.

## 🛠️ Mantenimiento

Este es un proyecto vivo. Iré actualizándolo y añadiendo nuevas plantillas a medida que las vaya creando o encontrando utilidades que merezca la pena guardar.
